# CSS
Интересные моменты CSS

## Общее

Задаём заголовки:

    h1, .h1 {
      размер
      отступ
      жирность
    }

## Селкторы

Контекстный селектор: ul li  
Прямой потомок: ul > li  
Уточняющий селектор: ul.menu  
Соседний селектор (абзац должен быть сразу): ul + p  
Соседний селектор (все абзацы): ul ~ p
